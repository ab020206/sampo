# WAD Practical Assignments (W1-W26)

This repository contains a complete set of Web Application Development (WAD) practical assignments, localized for the Indian context (PICT Pune).

## Project Structure
Each folder (`W1` through `W26`) represents a specific practical assignment.

### Technologies Used
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla, jQuery, jQuery Mobile), Bootstrap.
- **Backend**: Node.js, Express.js.
- **Database**: MongoDB (Local), JSON files.

## How to Run
1. **Frontend Projects**: Simply open the `index.html` file in any modern web browser.
2. **Backend Projects**:
   - Navigate to the project folder (e.g., `cd W20`).
   - Install dependencies: `npm install`.
   - Ensure MongoDB is running locally (`mongodb://localhost:27017`).
   - Start the server: `npm start` or `node server.js`.

## Localization
All projects have been localized to the Indian context, including:
- **Currency**: All prices are in Indian Rupees (₹).
- **Names/Cities**: Uses Indian student names and cities (Pune, Mumbai, etc.).
- **Institutional Context**: Refers to PICT (Pune Institute of Computer Technology).
